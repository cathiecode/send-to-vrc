Error! エラー!

何らかの原因によりアプリケーションが正常に動作していません。
アプリケーションを再起動しても解決しない場合、
再インストールなどをお試しください。 あるいは、手動で解決を試みることもできます。

The application is not functioning properly for some reason.
If restarting the application does not resolve the issue,
please try reinstalling it. Alternatively, you can try to resolve it manually.
